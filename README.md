# RetailFlow — SaaS POS System

A full-stack SaaS Point-of-Sale platform designed for retail stores 
and supermarkets, built with Java (Spring Boot) on the backend and 
React on the frontend.

## Features
- 🛒 Product & inventory management
- 💳 Billing, invoicing & payment processing
- 👥 Customer & loyalty management
- 📊 Sales reports & analytics dashboard
- 🏪 Multi-store / multi-branch support
- 🔐 Role-based access control (Admin, Cashier, Manager)
- ☁️ Cloud-hosted, accessible from any device

## Tech Stack
- **Backend:** Java, Spring Boot, Spring Security, REST APIs
- **Frontend:** React.js
- **Database:** MySQL / PostgreSQL
- **Auth:** JWT-based authentication
- **Deployment:** Docker / AWS
